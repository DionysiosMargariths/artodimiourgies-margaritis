# Αρτοδημιουργίες Μαργαρίτης

Επίσημη ιστοσελίδα για τον φούρνο **Αρτοδημιουργίες Μαργαρίτης** στα Καταλώνια Πιερίας.

## Live Website

https://artodimiourgies-margaritis.gr

## Περιγραφή

Η ιστοσελίδα παρουσιάζει την επιχείρηση, την ιστορία της, τα προϊόντα της, φωτογραφίες από τον φούρνο και βασικές πληροφορίες επικοινωνίας.

## Χαρακτηριστικά

- Responsive σχεδίαση για κινητά και υπολογιστές
- Gallery φωτογραφιών
- Modal προβολή εικόνων
- Ενότητες προϊόντων
- Σύνδεση με Google Maps
- SEO meta tags
- Open Graph εικόνα
- Favicon
- Sitemap και robots.txt

## Τεχνολογίες

- HTML5
- CSS3
- JavaScript
- Netlify
- GitHub
